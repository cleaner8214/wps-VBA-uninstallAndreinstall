# WPS VBA 7.1 官方修复包

## 问题描述
解决WPS打开包含VBA宏的文档时，弹出以下错误：
> The feature you are trying to use is on a CD-ROM or other removable disk that is not available.
> Insert the 'Microsoft Visual Basic for Applications 7.1 (x64)' disk and click OK.

## 个人版本
- 系统：win10 64位
- WPS教育版12.1.0.25222

## 使用方法
1. 关闭所有WPS程序窗口
2. 先使用卸载.bat对系统中的VBA文件进行卸载
3. 再右键点击.exe安装程序，选择"以管理员身份运行"
4. 按照安装向导完成安装
5. 重启WPS即可正常打开含宏文档
